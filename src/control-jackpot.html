Настройки Джекпота > uganda
Значение джекпота:
Накопленное значение джекпота на текущий момент времени. Это значение может отличаться от реального, так как джекпот сохраняется примерно каждые 60 минут.
Активация:
При достижении этого значения джекпот из пассивного состояния переходит в активное. В пассивном состоянии джекпот не может быть выигран. Когда джекпот в активном состоянии, у каждого игрока есть шанс выиграть джекпот при условии, что ставка игрока удовлетворяет условиям розыгрыша джекпота.
Таймер:
Время в секундах. После активации джекпота включается обратный таймер отсчета времени. Стартовое значение таймера равно величине поля Таймер. Шансы игрока выиграть джекпот увеличиваются при приближении таймера к нулю. Когда таймер кончается, джекпот будет гарантировано выигран первым же игроком, ставка которого удовлетворяет условиям розыгрыша джекпота.
Процент:
Процент с каждой ставки в джекпот. Значение процента может быть от 1 до 2 (целое число). 1 — рекомендуемое значение. Оно оптимально для залов с небольшим оборотом.
Макс. значение:
Если максимальное значение 0, оно игнорируется и джекпот выдается по таймеру. Если больше, чем значение активации, джекпот выдается по макс. значению. Чем ближе накопление к нему, тем выше вероятность выпадения джекпота.
Мин. ставка:
Минимальная ставка при которой игрок принимает участие в розыгрыше джекпота. При игре на меньшие ставки игрок выиграть джекпот не может.
Основное значение:
Основное значение джекпота, по умолчанию 0. Если значение больше нуля, после выпадения каждого джекпота, значение следующего джекпот будет стартовать с этой величины.
Внимание!!! Основное значение джекпота снимается с баланса зала
Сообщение:
Текст сообщения, которое будет выводится на игровом терминале. Допустимы только латинские символы, а также знаки «.», «,», «!», «:».
Время выпадения джекпота:
Вы можете регулировать временной интервал выпадения джекпотов. Минимальный промежуток — 1 час в сутки. Время задается по локальному времени зала.
